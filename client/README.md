# StayLux - Premium Hotel Booking Platform

A modern, feature-rich hotel booking application built with React, Vite, and Tailwind CSS. Featuring luxury hotel listings, seamless booking experience, and admin management dashboard.

## 🌟 Features

### User Features
- **Hotel Discovery**: Browse 15+ premium hotels across global destinations
- **Advanced Filters**: Filter hotels by price, rating, amenities, and location
- **Hotel Details**: Detailed hotel information with gallery, amenities, and room options
- **Booking System**: Smooth booking workflow with date selection and guest information
- **Checkout**: Secure payment integration with multiple payment methods
- **User Authentication**: Login/Register with profile management
- **Booking History**: View and manage all previous bookings
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Smooth Animations**: GSAP-powered page transitions and scroll effects

### Admin Features
- **Dashboard**: Real-time statistics and analytics
- **Hotel Management**: Add, edit, and manage hotel listings
- **User Management**: View and manage user accounts
- **Booking Management**: Track and manage all bookings
- **Revenue Analytics**: Monitor platform revenue and growth

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ and npm/yarn

### Installation

```bash
# Clone repository
git clone <repository-url>
cd client

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:5173`

### Production Build

```bash
npm run build
npm run preview
```
